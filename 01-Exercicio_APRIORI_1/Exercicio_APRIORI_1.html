Market Basket Analysis - Exercício 1¶

Para fazer o projeto de concluso de curso de Business Analytics da Data Science Academy (DSA) preciso entender como funciona o pacote APRIORI para fazer recomendaes de compras para uma base de clientes. Para isso, encontrei 2 artigos com o passo a passo da utilizao do algortimo. Nesse primeiro exerccio vou reproduzir o problema resolvido por Rizka Yolanda no seguinte artigo:

https://medium.com/@yolandawiyono98/market-basket-analysis-with-r-8001417a8e29

O artigo do Jie Wang foi base para compreenso dos objetos da classe transaction:

https://www.jdatalab.com/data_science_and_data_mining/2018/10/10/association-rule-transactions-class.html

Pacotes¶

Utilizarei o pacole arules, que cria as regras segundo o algortimo APRIORI, o pacote arulesViz, que permite a visualizao de objetos da classe transaction e o dataset Groceries, que j est no formato transaction e vem com o pacote arules.

Estrutura da classe transaction

Como explicado por Jie Wang, o objeto da classe transaction dividido em 3 componentes @data, @itemInfo e @itemsetInfo:

  • O @itemsetInfo uma dataframe vazio que s ser preenchido quando rodarmos a funo apriori
  • O @itemInfo um dataframe com 3 colunas: labels (que tem o nome de cada produto), level1 (que mostra um tipo de categorizao dos produtos) e level2 (que mostra um tipo de subcategorizao dos produtos)
  • O @data uma matriz que contm 2 listas: Groceries@data@i (lista completa dos itens comprados do primeiro ao ltimo cliente) e Groceries@data@p (lista com os ndices que dividem uma compra de outra).

Aplicação básica do apriori¶

Resumo dos parmetros:

  • Support (supp): a porcentagem mnima que um item aparece na lista de compras, para ser considerado uma regra.
  • Confidence (conf): a porcentagem mnima que um item aparece junto com outro item, para ser considerado uma regra.

# Definindo principais regras
rules = apriori(Groceries, parameter = list(supp = 0.001, conf = 0.8))

rules = sort(rules, by="confidence", decreasing=TRUE)

Fazendo recomendações¶

H duas perguntas importantes a serem respondidas:

  • Pergunta n 1 - O que os cientes costumam compram antes de comprar whole milk?
  • Pergunta n 2 - O que os clientes costumam comprar se eles compram whole milk?

Resposta da pergunta nº 1?¶

O que os cientes costumam compram antes de comprar whole milk?

Para responder a essa pergunta basta utilizar a funo apriori com os parametros do item que estamos procurando. Queremos congelar o item rhs e manter o lhs padro (veja o nome das colunas da tabela abaixo). Para responder a segunda pergunta faremos o oposto.

In [151]:
rules = apriori(data=Groceries, parameter=list(supp=0.005,conf = 0.5), 
                appearance = list(default="lhs",rhs="whole milk"),
                control = list(verbose=F))
rules = sort(rules, decreasing=TRUE,by="confidence")
inspect(rules[1:5])
